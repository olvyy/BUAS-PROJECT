# Ogres
is a Journey of the Prairie King (Stardew Valley) inspired arcade style shooter.

Try your best to survive as long as possible!

**WASD** - move 

**Arrow Keys** - Shoot 

**Escape** - quit

Power ups trigger automatically after pick up and do not stack.
